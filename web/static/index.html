配置管理

管理 Claude2API 系统运行参数、模型映射、网络代理与服务默认行为。

基础访问控制
API 调用密钥 OpenAI 兼容 API 的鉴权密钥；留空表示不修改。
管理页密码 修改后已有 Session 将立即失效，需要重新登录。
最大上下文长度 超过此长度将使用文件存储上下文。
内部重试次数 一次请求最多尝试多少个可调度 key；限流和可重试错误会换下一个。
单 Key 最大并发 每个 key 同时处理的请求数；建议保持 1，降低同一账号被打爆的概率。
全局最大并发 整个服务同时转发到 Claude 的请求数上限，超过后返回忙碌。
应用功能
自动删除对话 请求完成后自动删除 Claude 对话。
禁用角色前缀 不添加 User/Assistant 前缀。
禁用 Artifacts 在提示词中禁用 Artifacts 功能。
启用镜像 API 允许通过自定义前缀访问 API。
镜像 API 前缀 镜像 API 的 URL 前缀。
全局系统提示词覆盖 可为所有模型统一追加或替换系统提示词；模型级配置优先级更高。
